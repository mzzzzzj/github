1）Desktop 电脑桌面客户端：左上角三短线: 设置→隐私安全→开启"显示含有敏感内容的媒体"

2）macOS 电脑桌面客户端: 设置→隐私安全→开启"显示含有敏感内容的媒体"

然后重启客户端。
